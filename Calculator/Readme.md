# 🧮 Python Calculator

A simple command-line calculator built with Python that performs basic arithmetic operations through an interactive menu system. The calculator can handle multiple calculations in a single session while maintaining a history of previous results.

## ✨ Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 📜 Calculation History
* 🔄 Continuous Operation Loop
* 💻 Interactive Command-Line Interface
* ⚡ Fast and Lightweight

## 🎯 Project Goal

This project was created to practice core Python concepts and gain experience building interactive terminal applications.

## 📚 Concepts Used

* 🐍 Python Functions
* 🔀 Conditional Statements
* 🔄 While Loops
* 📥 User Input
* 📋 Lists
* 🧩 Basic Program Logic

## 📸 Preview

```text
Enter what you wanna do [sub mul div add exit] or [history]: add

Enter the first number: 10
Enter the second number: 5

15

Enter what you wanna do [sub mul div add exit] or [history]: history

15
```

## 🌟 Overview

The calculator allows users to select an operation, enter two numbers, and instantly receive the result. All completed calculations are stored in a history list that can be viewed at any time during the session.
