# ⚡ Multiplication Speed Test

🧠 A Python-based math quiz game that tests both multiplication skills and response speed. Players must answer randomly generated multiplication questions before the time limit expires.

## ✨ Features

- 🎲 Random multiplication questions
- ⏱️ Answer time tracking
- ⚡ Speed-based challenge system
- ✅ Correct answer detection
- ❌ Mistake tracking
- 📊 Accuracy calculation
- 🏆 Final performance summary
- 💻 Interactive terminal gameplay

## 🎯 Challenge Rules

- Answer 10 multiplication questions.
- Each answer must be submitted within 5 seconds.
- Answers submitted after 5 seconds are counted as slow responses.
- Final results include score, mistakes, slow answers, and overall accuracy.

## 🚀 Purpose

Built to improve Python fundamentals while creating an interactive game that combines mathematics, speed, and performance tracking.
