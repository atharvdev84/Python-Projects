# ✊📄✂️ Rock Paper Scissors Game

A simple terminal-based **Rock Paper Scissors** game built completely with code logic and core programming concepts. No APIs, frameworks, GUI libraries, or external tools were used — just pure programming fundamentals.

## 🚀 Features

* Player vs Computer gameplay
* Random computer choice generation
* Win / Lose / Draw detection
* Clean game logic
* Lightweight and beginner-friendly project

## 🛠️ Technologies Used

* Pure programming language only
* No APIs
* No GUI / UI frameworks

## 🎯 Purpose of This Project

This project was created to practice:

* Conditional statements
* Randomization logic
* User input handling
* Basic game development logic
* Problem-solving skills
