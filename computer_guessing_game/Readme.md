## 🎮 About This Project

* 🤖 A Python game where the computer tries to guess your number.
* 🔢 The user thinks of a number between a given range.
* 📈 The computer makes a guess and asks for feedback.
* ⬆️ Enter **"high"** if your number is higher.
* ⬇️ Enter **"low"** if your number is lower.
* ✅ Enter **"correct"** when the computer guesses the right number.
* 🧠 Uses the Binary Search technique to find the number efficiently.
* 🚀 Built to practice Python functions, recursion, and user input handling.
