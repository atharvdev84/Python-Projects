# 📂 File Organizer

A Python automation tool that organizes files into folders based on their file type. Select a folder, choose a format, and the program automatically moves files into their correct locations.

## ✨ Features

- 📄 Organize Text Files
- 🖼️ Organize Image Files
- 🎥 Organize Video Files
- 🎵 Organize Audio Files
- 📕 Organize PDF Files
- 📁 Organize All Supported Formats
- 🏷️ Custom Folder Names
- 🔄 Automatic File Movement
- 🛡️ Error Handling
- 📊 Displays Number Of Files Moved

## 🛠️ Technologies Used

- Python
- os
- shutil

## 📂 Supported Formats

**Text:** `.txt .doc .docx .rtf`

**Images:** `.jpg .jpeg .png .gif .bmp .webp`

**Videos:** `.mp4 .mov .mkv .avi .flv .wmv`

**Audio:** `.mp3 .wav .aac .flac .ogg`

**PDF:** `.pdf`

## 🚀 Overview

This project automatically sorts files into folders based on their format. It can organize individual file types or all supported formats at once, making folder management faster and easier.
