# 📏 Unit Conversion Quiz

🧠 A Python-based quiz game that tests unit conversion skills through randomly generated questions and a scoring system.

## ✨ Features

- 🎲 Random question generation
- 📏 Multiple unit conversions
- 🎯 Score tracking system
- ✅ Instant answer validation
- ⚠️ Input error handling
- 💻 Interactive terminal experience

## 🔄 Supported Conversions

- 📐 Centimeter ↔ Meter
- 📏 Meter ↔ Centimeter
- ⚖️ Gram ↔ Kilogram
- 🏋️ Kilogram ↔ Gram
- 📎 Centimeter ↔ Millimeter
- 📌 Millimeter ↔ Centimeter
- 🚗 km/h ↔ m/s
- 🏃 m/s ↔ km/h

## 📸 Preview

```text
Select a conversion:

1. cm ---> m
2. m ---> cm
3. g ---> kg

Enter choice: 1

45 cm to m = ?
Your answer: 0.45

✅ Correct!

Your score is 1
```
