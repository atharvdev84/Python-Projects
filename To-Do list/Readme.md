# 📋 To-Do List Manager

🗂️ A command-line task management application built with Python that allows users to create, organize, update, and track tasks using persistent JSON storage.

## ✨ Features

- ➕ Add new tasks
- ❌ Remove tasks
- 👀 View all tasks
- ✏️ Edit existing tasks
- ✅ Mark tasks as completed
- ⏳ Track pending tasks
- 🔍 Filter tasks by status
- 💾 Automatic JSON data storage
- 🛡️ Input validation and error handling
- 📂 Modular multi-file architecture

## 🚀 Project Overview

This project stores tasks in a JSON file, allowing data to persist between program sessions. Users can add, remove, edit, and manage task statuses while filtering completed and pending tasks. The project is structured across multiple Python modules, making the code more organized, maintainable, and scalable than a basic single-file application.
