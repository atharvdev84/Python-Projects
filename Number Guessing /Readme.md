# 🎮 Number Guessing Game 🔢

A simple Python game where the computer randomly selects a number between 1–100 and the player tries to guess it correctly.

## ✨ Features

🔹 Random number generation
🔹 Too High / Too Low hints
🔹 Score tracking
🔹 Error handling
🔹 Pure Python logic

## 🐍 Tech Used

`Python` • `random module` • `terminal`

## 🧠 What I Learned And Improved

✔ Loops
✔ Conditions
✔ Exception handling
✔ Game logic
