# 🎮 Word Scramble Game

🧩 A Python-based word puzzle game where players guess the original word from a scrambled version.

## ✨ Features

- 🎲 Random word selection
- 🔀 Dynamic word scrambling
- 🎯 Limited guess attempts
- 🏆 Win and Lose conditions
- 💻 Interactive terminal gameplay
- ⚡ Lightweight and beginner-friendly

## 📸 Preview

```text
Welcome to the Word Scramble Game!

Scrambled word: nraoge
Your guess: orange

🎉 Congratulations!
```

- 🎲 Random Module
